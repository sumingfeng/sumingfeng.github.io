通用提示层，可自定义标题、按钮添加、内容

点击出现弹窗提示，mold类型"tip"

HTML:

<a href="javascript:;" style="color:red;text-decoration: underline" id="p1">点击出现弹窗提示，mold类型"tip"</a>

CSS:

/*dialog*/

.popBox {
	background: #fff;
	border: 5px solid #0076bc;
	position: absolute;
	border-radius: 3px;
}
.popBox .phead h2 {
	color: #494949;
	font-size: 14px;
	background: url(../images/ui/line.png) bottom repeat-x;
	line-height: 50px;
	height: 50px;
	padding-left: 20px;
}
.popBox .close {
	background: url(../images/ui/icon_9.png) no-repeat;
	width: 26px;
	height: 26px;
	line-height: 999px;
	overflow: hidden;
	position: absolute;
	top: 3px;
	right: 3px;
	cursor: pointer;
}
.popBox .close:hover {
	background: url(../images/ui/icon_9.png) 0 -28px no-repeat;
}
.popBox .center {
	text-align: center;
}
.popBox .pbody {
	padding: 10px;
}
.popBox .button-area {
	padding: 15px 0;
	text-align: center
}
.popBox .button-area input {
	margin: 0 5px;
}

JS:

<script src=""../js/ui/su_dialog.js"></script>
<script>
	$(function() {

			$("#p1").click(function() {

				$.su_dialog(

					/* "tip" 为提示层标示 */

					"tip", {

						width: 400,

						height: "158",

						title: "自定义标题",

						tpl: '自定义可变的内容或提示信息',

						buttons: ["确定", "取消"],

						events: [

							function() {

								alert("确定");

								$.su_dialog_close(); //点确定之后关闭
							},

							function() {

								alert("取消");

								$.su_dialog_close(); //点取消之后关闭

							}

						]
					})

			})
		}
</script>


ajax请求同域下的页面

点击弹窗形式加载同域下的HTML页面,mold类型"ajax"

HTML:

<a href="javascript:;" style="color:red;text-decoration: underline" id="p2">点击弹窗形式加载同域下的HTML页面,mold类型"ajax"</a>

CSS:同提示层

JS:

<script src=""../js/ui/su_dialog.js"></script>
<script>
	$(function() {

	    $("#p2").click(function(e) {

				$.su_dialog("ajaxContent.html", {

						mold: "ajax", //加载类型

						title: "ajax弹窗标题",

						width: 400

					}

				)
			})
		}
</script>


iframe跨域请求页面

点击弹窗形式frame跨域加载页面,mold类型"iframe"

HTML:

<a href="javascript:void(0)" style="color:red;text-decoration: underline" id="p3">点击弹窗形式frame跨域加载页面,mold类型"iframe"</a>

CSS:同提示层

JS:

<script src=""../js/ui/su_dialog.js"></script>
<script>
	$(function() {
			$("#p3").click(function(e) {

				$.su_dialog("http://www.baidu.com", {

						mold: "iframe", //加载类型

						title: "iframe弹窗标题",

						width: 1000

					}

				)
			})
		}
</script>


将隐藏层的内容添加到弹窗

点击弹窗形式动态插入隐藏层,mold类型"content"

我是隐藏层内容！

HTML:

<a href="javascript:;" style="color:red;text-decoration: underline" id="p4">点击弹窗形式动态插入隐藏层,mold类型"content"</a>
<div id="hide_area" style="display:none">
 <p>我是隐藏层内容！</p>
</div>

CSS:同提示层

JS:

<script src=""../js/ui/su_dialog.js"></script>
<script>
	$(function() {
			$("#p4").click(function() {

				$.su_dialog("hide_area", {

						mold: "content", //加载类型

						title: "标题",

						width: 400

					}

				)
			})
		}
</script>
