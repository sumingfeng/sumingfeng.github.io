说明:

1.页面所有公共模块的样式都写在 base.css里，重构工程师只负责页面级的CSS编写。公用模块必须按照规范文档中的公共模块结构(HTML)编写。页面引入方式例如： 

<head>
	<meta charset="UTF-8">
	<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1">
	<title>Document</title>
	<link rel="stylesheet"  href="css/base.css">
	<link rel="stylesheet"  href="css/index.css">
 </head>

2.为了页面更好的扩展性，每个页面模块须定义额外的Class，这样既可以防止样式冲突，也便于代码复用和阅读。

各组件API:

表格组件：su_tableStyle.js
属性 功能
odd 定义奇数行的Class,如 $(".datatable").su_tableStyle({odd: "tr-odd"})
even 定义偶数行的Class
selected 定义选中行的Class
checkAll 定义全选复选框的ID
标签页组件：su_tabslide.js
属性 功能
trig

触发事件的标签，如$("#tab").su_tabslide({trig:".lis li",panel:".panel",current:"current"})

触发ID为tab下的，class为lis下的li元素，每个li元素对应的内容层class为panel,li选中状态class为current
panel 定义每个tag对应的内容层的Class
title 定义每个内容层切换的标题Class
current 定义标签当前状态
next 定义前进按钮的Class
prev 定义后退按钮的Class
autoPlay 布尔类型，说明是否自动播放，true为自动，false为停止
event 定义事件类型，如click、mouseover...
initTab 数字类型。定义初始化默认选中哪个标签，0代表第一个
delay 数字类型，定义自动播放间隔事件
弹窗组件：su_dialog.js
属性 功能
mold 类型标示符,默认为"tip"，有tip、ajax、iframe、content四种展现形式
width 数字型，表示弹窗的宽度
title 功能弹窗的标题
url 页面请求的地址，如果用在显示隐藏层，则表示隐藏层的ID，如果是提示层则用作标示符"tip"
method ajax请求的方法get/post
asnyc 布尔类型，表示同步或者异步，默认为false表示同步
tpl 弹窗的内容
buttons 弹窗的按钮，数组形式
events 每个按钮对应的事件函数，数组形式
callback 回调函数
分页组件：su_pagination.js
属性 功能
maxentries 总条目数，必选参数
items_per_page 每页显示的条目数，默认为10
num_display_entries 连续分页主体部分显示的分页条目数，默认为10
current_page 当前选中的页面，默认为0，表示第一页
link_to 分页的链接，默认是"#"
prev_text “前一页”分页按钮上显示的文字
next_text “下一页”分页按钮上显示的文字
ellipse_text 省略的页数用什么文字表示
prev_show_always 布尔类型，是否显示“前一页”按钮
callback 回调函数
表单组件：su_validation.js
属性 功能
required 布尔类型，如果为true,则为必填项
rangelength 字符长度限制区间，如[6,20]则表示6到20个字符
maxlength 数字型，最大长度
minlength 数字型，最小长度
email 布尔类型，如果为true，则表示验证开启
blank_card

布尔类型，银行卡验证。如果为true，则表示验证开启。该组件本身不带该功能，需扩展函数（见表单公共模块页JS）
phone 布尔类型，移动手机验证。如果为true，则表示验证开启。该组件本身不带该功能，需扩展函数（见表单公共模块页JS）
dateRule 布尔类型，日期格式验证（yy-mm-dd)。如果为true，则表示验证开启。该组件本身不带该功能，需扩展函数（见表单公共模块页JS）
url 布尔类型，网址验证。如果为true，则表示验证开启。
form 布尔类型，表示整个表单提交过程客户端验证是否通过
equalTo 表示两次输入是否一致；如密码确认equalTo: "#level_pwd"
tip组件：su_allTip.js
属性 功能
tpl 显示的主题内容母版
intoCon 布尔类型：移入消息框时是否显示
pos 提示位置：left\top\right\bottom
delay 信息框延时消息时间
showTime 消息框显示时间
url ajax请求url
method ajax请求method
data ajax请求data
